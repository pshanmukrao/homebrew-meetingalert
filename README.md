# homebrew-meetingalert

Homebrew tap for [MeetingAlert](https://github.com/AkhilaShanmukha/MeetingAlert) - A macOS menu bar app that displays your upcoming calendar meetings and automatically opens Zoom links.

## Installation

```bash
brew tap pshanmukrao/meetingalert
brew install meetingalert
```

## What is MeetingAlert?

MeetingAlert is a menu bar-only macOS app that:
- Shows your next/current meeting name in the menu bar
- Displays a dropdown list of upcoming events
- Automatically opens Zoom links when you click on meetings
- Shows full-screen alerts for meetings starting soon

## Requirements

- macOS 13.0 (Ventura) or later
- Calendar access permissions
